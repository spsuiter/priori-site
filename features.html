Prosecution Tools
From ODP to filing-ready draft.

IFW Collector

Connects directly to USPTO's Open Data Portal and pulls the complete Image File Wrapper for any application number. No manual downloading, no browser sessions, no ZIP extraction.

  • Live ODP API connection — data refreshed on demand
  • Automatic OA parsing: extracts claim list, rejections, examiner citations
  • Triggers Prosecution Intelligence lookup for the assigned examiner
  • Supports ODP June 2026 authenticated access (USPTO account required)
📥 IFW Collector — What It Pulls
  • Complete file wrapper (all documents)
  • Parsed rejection summaries (§101, §102, §103, §112)
  • Examiner name, art unit, and PTAB history
  • All cited prior art references
  • Prosecution timeline and status flags

Office Action Response Generator

Drafts a complete OA response — argument sections, claim amendments, and remarks — using your firm's prosecution history as worked examples. The AI never replaces attorney judgment; it accelerates it.

  • Examiner Intelligence panel: allow rate, avg OAs, pendency percentiles, top CPC codes
  • Firm Library RAG: retrieves relevant passages from uploaded firm briefs and responses
  • Corpus few-shot examples: your approved OA→Response pairs as drafting models
  • Supporting references: drag in technical papers or declarations for injection
  • Download ZIP: draft, claim markup, and IDS in one package
✍️ Generation Pipeline
  1. Parse OA → extract rejections
  2. Load examiner analytics from Prosecution Intel DB
  3. Retrieve firm corpus examples
  4. Query Firm Library (RAG)
  5. Inject supporting references
  6. Generate full draft response
  7. Attorney reviews, edits, signs

Corpus Manager Patent Pending

Stores approved OA→Response pairs as few-shot learning examples. The more quality pairs in the corpus, the more closely the AI reflects your firm's drafting style and argument strategies. Pairs can be imported directly from IFW or uploaded manually.

  • Approval workflow — pairs require attorney sign-off before entering corpus
  • 8-stage pipeline status dashboard for licensee onboarding
  • Per-client corpus segmentation for multi-licensee deployments
  • Corpus Maintenance: automated re-seeding and health monitoring
📚 Corpus Pipeline Stages
  • Import from IPMS or IFW
  • Deduplicate and normalize
  • Extract OA→Response pairs
  • Attorney approval review
  • Vectorize for retrieval
  • Inject into generation prompts
  • Monitor quality drift
  • Automated re-seed on schedule
Analysis Tools
Sharpen claims before they're filed.
🎯

Claim Scope Tuner Patent Pending

Systematically maps claim scope across six axes, helping practitioners find the optimal drafting position that maximizes protection while minimizing prosecution risk.

  • §101 eligibility risk assessment
  • §102/103 prior art exposure
  • §112 written description support
  • Infringement detectability scoring
  • Design-around resistance analysis
  • Invalidity challenge exposure
🔍

Outside Counsel Quality Review Patent Pending

Pre-filing screening for the issues that generate rejections, disclaimers, and post-grant challenges. Run before sending any draft to the USPTO or to in-house counsel.

  • Patent porn detection (20+ terms, with case law)
  • §112(f) nonce word identification
  • Antecedent basis cross-check across claim chain
  • Reference numeral gap analysis
  • AI: described-not-claimed / claimed-not-described
  • Inline annotated DOCX + structured report
👩‍💼

Inventor Review Patent Pending

Structured workflow for inventor review meetings. Translates legal claim language into plain English, validates disclosure coverage, and generates a sign-off checklist.

  • Claim-by-claim plain language translation
  • Disclosure cross-validation (AI-powered)
  • Amendment comparison for OA review meetings
  • Confirmation checklist — exportable DOCX
📐

Claim Element Analysis

Maps claim elements against prior art or accused product features. Generates a structured element-by-element breakdown for claim construction and infringement analysis.

🔬

Infringement Analysis

Attorney-only. Applies claim elements to accused product descriptions and generates a detailed infringement analysis including doctrine of equivalents considerations.

⚖️

Claim Construction

Generates intrinsic and extrinsic evidence summaries for claim terms. Flags prosecution history disclaimer risks and identifies terms requiring Markman treatment.

Prosecution Intelligence Patent Pending
Your firm's prosecution history, organized.

The Prosecution Intelligence database aggregates your firm's complete prosecution record and USPTO-wide benchmarks to inform every OA response with data your competitors don't have.

What It Captures

  • Complete firm prosecution history (IPMS import)
  • Live USPTO synchronization via ODP API
  • Per-examiner allow rates and argument analytics
  • Average OA counts, pendency P25/P75 percentiles
  • CPC code specialization by examiner
  • RCE/continuation propensity flags
  • USPTO-wide benchmarks (77,000+ examiners)

Where It Appears

  • Examiner Intelligence panel in OA Response Generator
  • Strategy Brief: recommended argument approach
  • Patent Application Generator: art unit-specific drafting
  • Portfolio Dashboard: firm-wide prosecution metrics
  • Prosecution Monitor: real-time status and alerts
