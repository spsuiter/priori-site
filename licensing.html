Licensing Model
How Priori licensing works.
🏢

Firm Deployment

Each license covers one firm installation on firm-owned hardware (typically a Mac Studio or comparable server). All attorneys and staff at the firm access Priori through a web browser — no per-seat installation.

🔑

Annual License

Priori is licensed annually. The license covers the full software platform, all tool modules, updates throughout the term, and access to the Administrator Guide for IT setup.

🚀

Onboarding Included

Licensing includes guided onboarding: Prosecution Intelligence database seeding from your practice management system, corpus bootstrapping from your existing file history, and staff training materials.

Pricing is per deployment. Priori is not priced per seat or per matter. A single firm license covers all users at the firm on one server. Contact us for a pricing proposal tailored to your firm size and matter volume.

Technical Requirements
What your firm needs to run Priori.

Priori runs on commodity hardware available at any electronics retailer. No data center, no cloud infrastructure.

Server (Recommended)

  • Apple Mac Studio (M2 or newer, 32GB+ RAM) — recommended
  • Or: Any modern macOS or Linux server with 16GB+ RAM
  • 30GB+ disk space for application, corpus, and databases
  • Connected to firm office network (wired recommended)

Client Requirements

  • Any modern web browser (Chrome, Safari, Edge, Firefox)
  • No software installation on attorney or staff machines
  • Network access to the firm server (office or VPN)

API Keys (Optional)

  • Anthropic (Claude API) — for highest-quality AI generation; optional if using local Ollama models only
  • USPTO ODP — for live IFW access; free registration at developer.uspto.gov
  • Microsoft 365 — for OTP login via firm email; required for multi-user access control
  • Ollama — free, runs fully locally; included in Priori deployment; no API key required
Local-only operation is supported. Ollama provides capable AI generation without any external API calls. Claude API is recommended for production OA responses but is not required.
