FAQ
Common questions.

Does client data ever leave our network?

No — unless you explicitly activate Cloud AI mode for a specific task. Ollama runs entirely locally. When Claude API is used, only your typed prompt text is transmitted — document content is never sent externally.

Can Priori file documents automatically?

No. Priori is a drafting and analysis tool. It never files anything. All work product is reviewed and signed by an attorney before going anywhere. This is by design.

What hardware do we need?

We recommend an Apple Mac Studio (M2 or newer, 32GB RAM). The server cost is a one-time capital expense separate from the software license. Staff access through any browser — no client installation needed.

How long does deployment take?

Typically 1–2 days for a technically capable IT administrator following the Deployment Checklist. Corpus seeding and database population can take an additional day depending on firm matter volume.

Do we need a Claude API key?

Not required. Ollama provides capable AI generation with fully local models. The Claude API is recommended for production OA responses and significantly improves output quality. Costs approximately $0.10–$0.40 per OA response at current rates.

Is this compatible with our practice management system?

Priori currently imports prosecution history via CSV export from [Practice Management System] and via direct USPTO ODP API. We are actively expanding IPMS integrations — ask about your specific system during the demo.
